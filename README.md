# MiniLibrary

This project is a simple Java console application created for the PGO Exercise 2 assignment.

## Project description

The program simulates a small library system.  

It includes books, readers, and library operations such as adding books, borrowing books, returning books, and displaying available books.

## Classes

- `Book` - stores information about a book and supports borrow/return operations

- `Reader` - stores reader data and borrowed books count

- `Library` - stores an array of books and supports search and borrowing logic

- `Main` - contains the test scenario

## Features

- create book objects

- create reader objects

- add books to the library

- print available books

- search a book by title

- count available books

- borrow a book

- return a book

## Repository structure

- `src/Book.java`

- `src/Reader.java`

- `src/Library.java`

- `src/Main.java`

## Test scenario

The `Main` class presents a simple scenario:

1. create books and a reader

2. add books to the library

3. print available books

4. borrow one book

5. print the state again

6. return the book

7. print the final state
